# 0212_WSOP TV OTT 플랫폼 선정

OTT 외주 업체 비용 비교 | 2026-02-12 | $1 = ₩1,450

---

## 외주 업체 선정 과정 개요

국내외 OTT 업체 5곳에 RFP를 발송하고 미팅 및 견적을 진행했습니다.

AWS Direct는 비용과 일정이 미확정이고 내부 개발팀이 전제되어 제외했습니다. 메가존클라우드는 구축 기간이 2년 이상이라 제외했습니다.

Vimeo OTT, Brightcove+Accedo, 카테노이드 위주로 비교 자료를 작성했습니다. 기본 OTT 기능은 세 곳 모두 차이가 없습니다.


---

## 비용 비교

세 업체의 접근 방식이 다릅니다.

- **Vimeo OTT**: SaaS. 싸고 자유도 낮음.
- **Brightcove+Accedo**: PaaS 커스텀 빌드. 비싸고 자유도 높음.
- **카테노이드**: 솔루션 + MAM 통합. 소스코드 이관 포함.

### 구축비에 대해서

| 업체 | 구축비 | 산정 범위 |
|------|--------|----------|
| Vimeo OTT | **$ 60,000** | SaaS 템플릿 설정 |
| 카테노이드 | **~$ 407,000** | OTT 단독 (MAM ~$ 621,000 별도) |
| Brightcove+Accedo | **$ 650,000** | 7개 플랫폼 커스텀 빌드 |

격차가 극단적으로 보입니다. 하지만 산정 범위가 업체마다 다릅니다. Vimeo는 SaaS 템플릿 설정이고 Brightcove는 7개 플랫폼 커스텀 빌드입니다. 같은 구축비로 단순 비교하면 오해를 만듭니다.

### 커스터마이징 범위에 대해서

구축비 격차는 커스터마이징 범위에서 옵니다.

**Vimeo OTT**는 템플릿 수준의 변경만 가능합니다.

- 로고, 스플래시 화면, 컬러 테마
- 영역 사이즈, 폰트색, 바탕색
- CSS 제어 가능 범위
- 제공 레이아웃 외 커스터마이징 불가

![Vimeo OTT 템플릿 인터페이스](images/vimeo-ott-browse.png)

위 템플릿에서 변경할 수 있는 것이 거의 없습니다. **추가 개발 비용이 발생하지 않습니다.**

**Brightcove+Accedo**는 Brightcove가 백엔드를, Accedo가 프론트 에디터를 담당합니다.

- 페이지 구조 편집, navigation 구성
- 콘텐츠 rail / playlist 구성
- 실시간 preview, drag-and-drop UI 커스터마이징
- 커스텀 API 연동 컴포넌트 추가 가능

![Brightcove+Accedo 커스텀 OTT 앱](images/brightcove-ott-full.png)

7개 플랫폼에 각각 커스텀 UI를 구성할 수 있습니다. **구축비 $ 650,000은 이 자유도에 대한 비용입니다.**

**카테노이드**는 Kollus 솔루션 기반으로 OTT를 구축합니다. 소스코드 이관이 포함되어 론칭 후 자체 수정이 가능합니다. OTT와 MAM을 하나의 업체에서 통합 관리할 수 있다는 것이 차별점입니다.

### 운영비에 대해서

Vimeo와 Brightcove는 Tier별 견적이 확보되어 있습니다.

| Tier | Vimeo OTT (고정 Bandwidth형) | Brightcove+Accedo (Traffic 과금형) |
|---|---|---|
| **S** (70TB / 5PB) | **$ 296,000** | **$ 396,000** |
| **L** (500TB / 15PB) | **$ 425,000** | **$ 532,000** |
| **XXL** (1PB / 20PB) | **$ 456,000** | **$ 583,000** |

카테노이드는 연간 ~**$ 187,000**입니다 (MAM 라이선스 포함, MAU 10,000 · CDN 25TB/월 기준).

과금 형태가 다르다는 것을 염두에 두어야 합니다. Vimeo는 Bandwidth 고정형이고 Brightcove는 Traffic 비례형입니다. 카테노이드가 가장 낮아 보이지만 소규모 기준이라 단순 비교는 어렵습니다. (트래픽 예측 후 정밀 비교를 진행하겠습니다)

소스코드 소유권은 카테노이드만 이관이 포함됩니다. Vimeo는 SaaS 특성상 불가하고 Brightcove는 협의 가능합니다. MAM은 카테노이드 Loomex만 포함됩니다. 나머지 두 업체를 선택하면 MAM은 별도 확보가 필요합니다.

## 중간 정리

기준에 따라 선택지가 달라집니다.

**Vimeo OTT 선택**

- 구축비 최소화가 우선이다.
- 3개월 내 론칭이 필요하다.

**카테노이드 선택**

- OTT와 MAM을 통합 관리하고 싶다.
- 소스코드 이관으로 장기적 자체 운영을 계획한다.
- 구축비 ~$ 1,028,000을 감수할 수 있다. (OTT + MAM)

**Brightcove+Accedo 선택**

- 7개 플랫폼의 커스텀 UI/UX가 필요하다.
- 구축비 $ 650,000을 감수할 수 있다.
- 6~8개월 개발 기간을 감수할 수 있다.

**의견**

운영비는 초기에 크게 차이가 나지 않을 거라고 판단합니다. 카테노이드가 저렴해 보이지만 소규모 기준이고, Vimeo도 Bandwidth 한도를 초과하면 상위 과금 체계로 올라가야 합니다. 결국 운영비는 비슷해 질 것으로 보입니다. (트래픽 예측 후 정밀 계산을 진행하겠습니다)

관건은 구축비입니다.

커스텀 UI/UX가 브랜드 차별화에 필수적이라면 Brightcove+Accedo로 가도 됩니다. 하지만 기본 OTT 론칭이 우선이고 추후 커스터마이징을 검토할 예정이라면 구축비를 아끼고 Vimeo OTT로 시작하는 것이 좋다고 판단됩니다.

어느 OTT를 선택하든 MAM은 별도 확보가 필요합니다. 카테노이드 Loomex가 검토 대상입니다.
