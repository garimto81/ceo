# 구조 검증 체크리스트

PRD-001 v2.0 섹션 2 기반 구조 검증.

---

## 자동 검증 항목

### 제목 계층 (PRD 섹션 2.2)

- [ ] **[AUTO-1]** H1이 3개 이하인가
  - 검증 방법: 정규식 `^# ` 카운트
  - 기준: 최대 3개

- [ ] **[AUTO-2]** H2가 H1당 3개 이하인가
  - 검증 방법: H1-H2 계층 파싱
  - 기준: 각 H1 섹션 내 H2 최대 3개

### 메타 정보 (PRD 섹션 2.5)

- [ ] **[AUTO-22]** 메타 정보가 첫 2줄에 있는가
  - 검증 방법: 첫 2줄에 제목+요약 확인
  - 기준: 제목(1줄) + 1줄 요약

### 여백 패턴 (PRD 섹션 13)

- [ ] **[AUTO-21]** 여백 패턴이 적절한가
  - 검증 방법: 빈 줄 패턴 분석
  - 기준: 3단계 여백 시스템
    - H1/H2 전후: 대주제 전환
    - 이중 여백: 소주제 간 대전환
    - 단일 여백: 문단 간 호흡 (3-5문장마다)

## 수동 검증 항목

### 5단계 구조 (PRD 섹션 2.1)

- [ ] **[MANUAL-1]** 5단계 구조를 따르는가
  - 판단 기준: 과정 요약-핵심 비교-중간 정리 순서
  - 확인 사항:
    - 1단계: 과정 요약 (어떻게 여기까지 왔는가)
    - 2단계: 핵심 비교 (두 선택지의 본질적 차이, H2로 세분화)
    - 3단계: 중간 정리 (조건별 선택지 + 의견)

### 팩트/의견 비율 (PRD 섹션 2.3)

- [ ] **[MANUAL-2]** 팩트/의견 비율이 70%/30%에 가까운가
  - 판단 기준: 문단별 팩트/의견 분류 후 비율 계산
  - 허용 범위: 60-80% 팩트, 20-40% 의견
