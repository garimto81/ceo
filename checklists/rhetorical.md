# 수사 검증 체크리스트

PRD-001 v2.0 섹션 3, 4, 5 기반 수사 구조 검증.

---

## 자동 검증 항목

### 접속사 빈도 (PRD 섹션 7.2)

- [ ] **[AUTO-5]** "하지만"이 3회 이하인가
  - 검증 방법: 정규식 `하지만` 카운트
  - 기준: 최대 3회

- [ ] **[AUTO-6]** "결국"이 3회 이하인가
  - 검증 방법: 정규식 `결국` 카운트
  - 기준: 접속사 2회 + 문장 내 부사 1회 = 총 3회

- [ ] **[AUTO-18]** "다만"이 1회 이하인가
  - 검증 방법: 정규식 `다만` 카운트
  - 기준: 최대 1회 (불확실성 완충용)

### 금지 접속사 (PRD 섹션 7.2, 9)

- [ ] **[AUTO-12]** "다음으로", "이어서" 전환 문구가 없는가
  - 검증 방법: 정규식 `다음으로|이어서`
  - 기준: 0회

- [ ] **[AUTO-13]** 문어체 접속사가 없는가
  - 검증 방법: 정규식 `그러나|그렇지만|그럼에도 불구하고`
  - 기준: 0회 (구어체 "하지만" 사용)

## 수동 검증 항목

### 3단 반전 패턴 (PRD 섹션 3.1, 3.2)

- [ ] **[MANUAL-3]** "인상 - 하지만 - 심층 - 결국 - 수렴" 구조가 최소 1회 존재하는가
  - 판단 기준: 반전 구조의 논리적 완성도
  - 확인 사항:
    - 표면적 인상 제시
    - "하지만"으로 반전
    - 심층 분석 제공
    - "결국"으로 수렴
    - 최종 판단
  - 선제적 이의 해소: 독자의 직관을 먼저 인정한 뒤 반전

### Pivot 문장 (PRD 섹션 3.3)

- [ ] **[MANUAL-9]** Pivot 문장이 주제 전환 역할을 수행하는가
  - 판단 기준: 특정 문장이 서술 방향을 완전히 전환
  - 확인 사항:
    - 기존 프레임을 뒤집거나
    - 논의 초점을 이동하는 문장 존재

### 양보-회수 패턴 (PRD 섹션 3.4)

- [ ] **[MANUAL-10]** 양보-회수 패턴이 논리적으로 타당한가
  - 판단 기준: 허용-조건-회수의 논리적 타당성
  - 확인 사항:
    - 반대 선택지를 먼저 허용
    - 조건절로 제한
    - 추천 선택지로 회수하여 강화

### 종결어미 (PRD 섹션 4.1, 4.4)

- [ ] **[MANUAL-5]** 종결어미 3단계 구분이 정확한가
  - 판단 기준: 사실/추정/판단의 맥락 일치
  - 확인 사항:
    - 사실: ~했습니다, ~입니다 (과거 사실, 단정)
    - 추정: ~보입니다, ~것으로 보입니다 (인상, 예측)
    - 판단: ~판단합니다, ~판단됩니다 (결론, 추천)
    - 강한 확신: ~싶습니다 (1-2회)

- [ ] **[MANUAL-14]** 종결어미 점층 배치가 적절한가
  - 판단 기준: 사실→추정→판단 순서
  - 확인 사항:
    - 문서 전반부: 사실 위주
    - 중반: 추정 교차
    - 후반: 판단 집중

### 설득 아크 (PRD 섹션 14)

- [ ] **[MANUAL-8]** 설득 아크 7단계 흐름이 자연스러운가
  - 판단 기준: 7단계 흐름의 자연스러움
  - 확인 사항:
    1. 과정 설명 (신뢰 구축)
    2. 비유 프레임 설치
    3. 첫 pivot (전환)
    4. 비용 반전 ("비싸 보입니다. 하지만")
    5. 중립화 ("둘의 단순 비교는 어렵습니다")
    6. 1차 추천
    7. 킬링 블로우 + 최종 판단

### 비유 (PRD 섹션 5.1)

- [ ] **[MANUAL-4]** 비유가 적절한가
  - 판단 기준: 한국 독자 친화적 + 비교 대상 대응 정확성
  - 확인 사항:
    - 도입부에 한 번만 사용
    - 비교 쌍이 명확
    - 본문의 비교 대상과 정확히 대응

### 재귀적 의견 구조 (PRD 섹션 5.4)

- [ ] **[MANUAL-13]** 의견 내부에 반전-수렴 패턴이 존재하는가
  - 판단 기준: 의견 내 반전-수렴 패턴
  - 확인 사항:
    - 1차 판단 → 양보 → 반전 → 재수렴 → pivot → 최종 판단

### 문장 길이 수사 (PRD 섹션 7.3)

- [ ] **[MANUAL-12]** 문장 길이가 정보 밀도에 맞게 조절되었는가
  - 판단 기준: 장문(분석)/단문(판단) 배치
  - 확인 사항:
    - 복잡한 분석: 장문 (100자 이상)
    - 핵심 판단: 단문 (30자 이하)
